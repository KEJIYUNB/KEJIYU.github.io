# 科技鱼 · 个人站

这是我的个人网站源码，基于 GitHub Pages 构建，用来展示我的项目、代码以及一些有趣的折腾内容。

## 🌐 访问网站

👉 https://kejiyu.xyz

---

## ✨ 项目介绍

本网站是一个偏向 **个人展示 + 实验性质** 的主页，主要用于：

- 展示我的 GitHub 项目
- 分享一些有趣的网页效果
- 记录折腾过程和想法
- 放一些小工具 / 小玩具

整体风格偏 **赛博风 / 科技感 / 动效丰富**。

---

## 🧩 页面功能

当前网站包含（持续更新中）：

- 🏠 首页展示（动态背景 + 光效 + 卡片动效）
- ⌨️ 打字机动态效果
- 🧠 3D 卡片交互（鼠标倾斜效果）
- ✨ 粒子背景（particles.js）
- 🕹️ 小游戏（如俄罗斯方块）
- 🔗 外部脚本 / 工具入口

---

## 🚀 技术栈

本项目主要使用：

- HTML5
- CSS3（大量动画 / 渐变 / 光效）
- 原生 JavaScript
- particles.js（粒子背景）

无框架，纯手写，方便自由折腾。

---

## 📦 部署方式

本项目基于 **GitHub Pages** 部署：

1. Fork / Clone 本仓库
2. 修改 `index.html`
3. 推送到 GitHub
4. 在仓库设置中开启 Pages
5. 绑定域名（可选）
